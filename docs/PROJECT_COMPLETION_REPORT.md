## 1. Project Overview

Athena ECM (Enterprise Content Management) is a modern, scalable, and intelligent document management system. Over the course of 6 sprints, we have built a comprehensive backend platform that rivals commercial solutions in core capabilities.

## 2. Architecture & Technology Stack

The system is built on a microservices-ready architecture:

*   **Core Backend**: Java 17, Spring Boot 3.2
*   **Database**: PostgreSQL 15 (JSONB for metadata)
*   **Search Engine**: Elasticsearch 8.11 (Full-text & Faceted search)
*   **Process Engine**: Flowable BPMN 7.0
*   **Storage**: MinIO (S3 compatible)
*   **AI/ML**: Python FastAPI (Sidecar service)
*   **Message Queue**: RabbitMQ (Event bus)
*   **Cache**: Redis 7

## 5. Deployment Guide

The entire stack is containerized. To start:

```bash
# 1. Clone repository
git clone https://github.com/Athena/ecm-core.git

# 2. Build services
cd ecm-core && mvn clean package
cd ml-service && docker build -t ml-service .

# 3. Start infrastructure
docker-compose up -d

# 4. Access
# API Docs: http://localhost:8080/swagger-ui.html
# Frontend: http://localhost:3000
```
