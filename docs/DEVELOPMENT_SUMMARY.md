## System Architecture

The final delivered system consists of:

1.  **Core Backend (Java/Spring Boot)**:
    *   Manages Metadata, Security, Versions, and Search.
    *   Hosts the **WOPI** and **WPS** integration endpoints.
2.  **Frontend (React)**:
    *   **File Browser**: Full feature grid with drag-and-drop.
    *   **Editor Page**: Dedicated route `/editor/:id` embedding the external editor iframe.
3.  **AI & Automation**:
    *   **ML Service**: Auto-classifies content in the background.
    *   **Workflow Engine**: Drives approval processes.

## Deployment Checklist

1.  **Configuration**:
    *   Update `ecm.wps.*` in `application.yml` with valid AppID/AppKey from WPS Open Platform.
    *   Ensure `ecm.api.base-url` matches the public-facing URL (for callbacks).
2.  **Infrastructure**:
    *   Run `docker-compose up -d --build`.
    *   Configure Nginx/Load Balancer for SSL (HTTPS is required for most Office Online integrations).
