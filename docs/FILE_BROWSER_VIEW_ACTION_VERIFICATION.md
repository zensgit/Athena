# File Browser View Action - Verification

Date: 2025-12-22

## Automated Checks
- `npm run lint` (ecm-frontend)

## Manual Verification Steps
1. Open the file browser and right-click a PDF document.
2. Confirm a "View" option is present.
3. Click "View" and confirm the preview dialog opens.
4. For an Office document, confirm both "View" and "Edit Online" are available.
5. Double-click a document row/card and confirm the preview dialog opens.

## Notes
- Manual verification requires a running UI + API.
